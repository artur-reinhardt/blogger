HTML: Lenguaje de etiquetas de hipertexto

HTML  es el componente más básico de la Web.  Define el significado y la estructura del contenido web.  Junto con HTML tambien se utilizan otras tecnologías para describir la apariencia y presentación de una página web  CSS  la funcionalidad y el comportamiento  JavaScript

Los  Elementos  de HTML

<html>

El elemento  <html>  (o elemento HTML raíz) representa el root de un documento HTML. El resto de elementos descienden de él.  Tiene como contenido permitido un elemento  <head>  seguido de un elemento  <body>

<!DOCTYPE html>
<html>
  <head>...</head>
  <body>...</body>
</html>

<head>

El elemento  <head>  provee información general (metadatos) acerca del documento, incluyendo su título y enlaces a scripts y hojas de estilos.  Tiene como contenido permitido uno o más elementos de metadatos donde exactamente uno es  <title>  Si el elemento es un  srcdoc  de un  <iframe>  o si la información de título está disponible desde un protocolo de nivel superior,  puede contener cero o más elementos de metadatos.

<html>
 <head>
   <title>Título del documento</title>
 </head>
</html>
